Default notation/terminology conventions:

- The natural numbers include 0.
- 2D coordinates are labelled as follows:
  ```
  (0, 0) (1, 0) (2, 0) ...
  (0, 1) (1, 1) (2, 1) ...
  (0, 2) (1, 2) (2, 2) ...
  ...    ...    ...    ...
  ```
